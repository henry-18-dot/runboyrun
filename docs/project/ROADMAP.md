# Roadmap / 路线图

## Now / 当前

- 保持 v0.1.0 technical skeleton 可运行。
- 建立 Version Pack、决策记录、开放问题和五助手入口。
- 使用现有本地 asset review pipeline 管理候选资产。

## Next / 下一步

- 锁定 visual asset taxonomy：biome、layer、role、status、tags。
- 让五助手输出固定为摘要、风险、建议、问题和 Codex-ready tasks。
- 在不改游戏核心代码的前提下，补齐第一批可审查 visual candidates。

## Later / 以后

- Asset Orchestrator v1：优先 visual workflow preset。
- ComfyUI 本地 watcher 或 API runner，根据开放问题决策落地。
- Real asset integration：把 approved 视觉资产接入 renderer，把 approved 音频接入 audio managers。
- v0.5 阶段再推进更完整的 audio/music integration。

## Not Now / 现在不做

- 不做云端审查页面。
- 不做自动音频生成。
- 不让 AI 生成资产绕过人工 review。
- 不引入大型游戏引擎或重构运行时架构。
